Single number I

a^a == 0, if I know this, this problem would not be hard. {% highlight cpp linenos %} class Solution { public: int singleNumber(int A[], int n) { int sum = 0; for ( int i = 0; i < n; i++ ) { sum ^= A[i]; } return sum; } }; {% endhighlight %}

Single number II

This is a little harder. Use bit operation to solve this question. Assuming one, two, three stand for numbers apperars one, two and three times. Check two first, then check one and three. {% highlight cpp linenos %} class Solution { public: int singleNumber(int A[], int n) { int one = 0; int two = 0; int three = 0; for ( int i = 0; i < n; i++ ) { two |= one&A[i]; one = one^A[i]; three = one&two; two &= ~three; one &= ~three; } return one; } }; {% endhighlight %}
