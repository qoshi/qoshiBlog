Good problem

We always want to resolve the problem from head to tail. This time we need to solve this problem from back to save time complexity. {% highlight cpp linenos %} class Solution { public: void merge(int A[], int m, int B[], int n) { if ( n == 0 ) { return; } int k = m+n-1; int i = m-1; int j = n-1; while ( j >= 0 ) { if ( i >= 0 && A[i] > B[j] ) { A[k--] = A[i--]; } else { A[k--] = B[j--]; } } } }; {% endhighlight %}
