I not hard

Use two pointer i = 0 and j = 1.If they are the same , pointj to the next. Otherwise, put A[j] to the next of i, then point j to the next. {% highlight cpp linenos %} class Solution { public: int removeDuplicates(int A[], int n) { if ( n <= 1 ) { return n; } int i = 0; int j = 1; while ( j < n ) { if ( A[i] == A[j] ) { j++; } else { A[++i] = A[j++]; } } return i+1; } }; {% endhighlight %}

II not hard too

Use some a valiable to record the times A[i] and A[j] are same.Then do the same thing.Attention that when they are same and mark is not reach 1, donot forget set A[++i] with A[j++]. {% highlight cpp linenos %} class Solution { public: int removeDuplicates(int A[], int n) { if ( n <= 2 ) { return n; } int mark = 0; int i = 0; int j = 1; while ( j < n ) { if ( A[i] == A[j] ) { if ( mark < 1 ) { A[++i] = A[j++]; mark++; } else { j++; } } else { A[++i] = A[j++]; mark = 0; } } return i+1; } }; {% endhighlight %}
